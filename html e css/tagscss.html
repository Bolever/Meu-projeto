Tags que estou estudando até agora


border

  • border-width → Define a espessura da borda (ex: 1px, 5px).
  • border-style → Define o estilo da borda (solid, dashed, dotted etc.).
  • border-color → Define a cor da borda (red, blue, #333 etc.).
  • border-radius → Arredonda os cantos do elemento (10px, 50%).
  • border-top → Controla a borda superior separadamente.
  • border-right → Controla a borda direita separadamente.
  • border-bottom → Controla a borda inferior separadamente.
  • border-left → Controla a borda esquerda separadamente.

height

  • height → Define a altura de um elemento (100px, 50%, auto).
  • min-height → Define a altura mínima que um elemento pode ter.
  • max-height → Define a altura máxima que um elemento pode ter.

width

  • width → Define a largura de um elemento (100px, 50%, auto).
  • min-width → Define a largura mínima que um elemento pode ter.
  • max-width → Define a largura máxima que um elemento pode ter.

padding

  • padding → Define o espaçamento interno entre o conteúdo e a borda de um elemento (ex: 10px, 1em).
  • padding-top → Define o espaçamento superior.
  • padding-right → Define o espaçamento à direita.
  • padding-bottom → Define o espaçamento inferior.
  • padding-left → Define o espaçamento à esquerda.

font

  • font-family → Define a família da fonte (ex: Arial, Verdana).
  • font-size → Define o tamanho da fonte (ex: 16px, 1em, larger).
  • font-weight → Define o peso da fonte (ex: normal, bold, lighter).
  • font-style → Define o estilo da fonte (ex: normal, italic, oblique).
  • font-variant → Define a variante da fonte (ex: normal, small-caps).
  • font-stretch → Controla a espessura e largura da fonte (ex: normal, condensed).

color

  • color → Define a cor do texto (ex: red, #333, rgb(0, 0, 255)).
  • color-opacity → Controla a opacidade da cor do texto (ex: rgba(255, 0, 0, 0.5)).

background

  • background-color → Define a cor de fundo do elemento (ex: red, #f0f0f0).
  • background-image → Define uma imagem de fundo (ex: url('imagem.jpg')).
  • background-position → Define a posição da imagem de fundo (ex: top left, center center).
  • background-size → Define o tamanho da imagem de fundo (ex: cover, contain, 100px 200px).
  • background-repeat → Define se a imagem de fundo vai se repetir (ex: no-repeat, repeat, repeat-x).
  • background-attachment → Define se a imagem de fundo deve rolar com o conteúdo ou ficar fixa (ex: scroll, fixed).
  • background-clip → Controla onde o fundo é desenhado (ex: border-box, padding-box).
  • background-origin → Define a origem do fundo (ex: padding-box, content-box).

margin

  • margin → Define o espaçamento externo entre o elemento e seus vizinhos (ex: 10px, 2em).
  • margin-top → Define o espaçamento superior.
  • margin-right → Define o espaçamento à direita.
  • margin-bottom → Define o espaçamento inferior.
  • margin-left → Define o espaçamento à esquerda.

box-shadow

  • box-shadow → Adiciona uma sombra ao redor de um elemento (ex: 2px 2px 5px rgba(0, 0, 0, 0.3)).

Parâmetros:

  • horizontal offset → Deslocamento horizontal da sombra (ex: 2px).
  • vertical offset → Deslocamento vertical da sombra (ex: 2px).
  • blur radius → Definição de desfoque da sombra (ex: 5px).
  • spread radius → Definição do tamanho da sombra (ex: 0px).
  • color → Cor da sombra (ex: rgba(0, 0, 0, 0.3)).

text-shadow

  • text-shadow → Adiciona uma sombra ao texto.

Parâmetros:

  • horizontal offset → Deslocamento horizontal da sombra (ex: 2px).
  • vertical offset → Deslocamento vertical da sombra (ex: 2px).
  • blur radius → Definição de desfoque da sombra (ex: 4px).
  • color → Cor da sombra (ex: rgba(0, 0, 0, 0.5)).

text

  • text-align → Define o alinhamento do texto (ex: left, center, right, justify).
  • text-decoration → Adiciona ou remove uma decoração no texto (ex: underline, line-through, overline, none).
  • text-transform → Controla a transformação do texto (ex: uppercase, lowercase, capitalize).
  • text-indent → Define o recuo do texto (ex: 20px, 2em).
  • text-shadow → Adiciona uma sombra ao texto (ex: 2px 2px 4px rgba(0, 0, 0, 0.5)).
  • letter-spacing → Controla o espaçamento entre as letras (ex: 1px, 0.1em).
  • line-height → Define a altura da linha do texto (ex: 1.5, 20px).
  • white-space → Controla o comportamento do texto quanto a quebras de linha e espaços (ex: normal, nowrap, pre).

Importar fontes

  • Para importar uma fonte externa, você pode usar a regra @import no CSS ou a tag <link> no HTML.
  • O @font-face permite que você defina fontes personalizadas e as use diretamente no seu projeto, carregando o arquivo da fonte do seu servidor ou do diretório local.
