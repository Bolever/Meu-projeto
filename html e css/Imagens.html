Favicon

Favicon é um ícone pequeno associado a uma página da web, que aparece na aba do navegador ou na lista de favoritos. Ele ajuda a identificar visualmente o site.


        <head>
        <link rel="icon" href="caminho/para/o/favicon.ico" type="image/x-icon">
        </head>

Imagens

A tag <img> é usada para exibir imagens em uma página HTML. Ela é uma tag auto-fechada, ou seja, não precisa de uma tag de fechamento. Você define o caminho da imagem com o atributo src e pode adicionar outros atributos como alt para descrição, width e height para dimensões. 
<img src="imagem.jpg" alt="Descrição da imagem" width="300" height="200">
